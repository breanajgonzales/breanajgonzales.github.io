Volunteers and teachers can then search and connect.

  • Volunteer backend developer assisting with design and implementation of API for mobile team.
  • Coded in Java and Spring. Created builds using Maven.

  • Implemented basic CRUD functionality for banners, needs and offerings, logic for volunteers to "accept" needs and offerings, etc.

  • Client: STEM Action Center
